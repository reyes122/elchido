Proposito del Software

Visual Studio Code es un editor de código fuente desarrollado por Microsoft. Es una herramienta de desarrollo de software muy popular que se utiliza para crear y editar código en varios lenguajes de programación, como C++, C#, Python, Java, JavaScript, HTML, CSS y muchos otros.

Proceso de Instalación

Para instalar Visual Studio Code, deberas:

  1. Descargar Visual Studio Code -> Descargar
  2. Abrir archivo de instalación, seguir instrucciones (dar siguiente)
  3. Abrir VSC
  4. Persoalizar con las extensiones necesarias
  5. Listo, ahora a trabajar con VSC
