Proposito del Software

Es un software de control de versiones, pensado para la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando tienen muchos archivos fuente

Proceso de Instalación

Para instalar Git, deberas:

  1. Descargar Git -> Descargar
  2. Abrir archivo de instalación, seguir instrucciones
  3. Seleccionas Donde Guardar los archivos
  4. Configuras la instalacion de acuerdo a lo que requieras
  5. Das click en Instalar
  6. Te mostrara la Terminal
